# Overview

## What the module does

**wp-module-next-steps** manages next steps for customers in Newfold brand plugins (e.g. Bluehost, HostGator). It provides plan-based task lists (tracks → sections → tasks) with smart task completion, a REST API, and React UI (widget and portal). It integrates with the Onboarding module (site type) and Solutions module (recommendations).

## Who maintains it

- **Newfold Labs** (Newfold Digital). Distributed via Newfold Satis.
